01 // SYSTEM_ARCHITECTURE

Hardware & Drive-by-Wire

Automama is our in-house Autonomous Ground Vehicle (AGV) project, designed to transform a custom-built electric vehicle (EV) into a fully programmable robotic platform. The EV chassis was enhanced with sensor feedback systems for braking, steering, and acceleration, enabling complete drive-by-wire control.

Perception Pipeline

We integrated a stereo vision system using an IMX219 stereo camera pair. Frames from the left camera are processed through a YOLOv8n-Segmentation model to segment roads and obstacles. These segmentation masks guide stereo depth estimation, providing precise depth information for drivable areas and obstacles.

Mapping & Navigation

Using depth maps, we construct a 2D occupancy grid of the road. A custom local planner detects and follows the largest navigable gap in real time, ensuring safe short-term path planning and allowing Automama to react dynamically to changing environments. Our long-term vision is to map the entire university campus and integrate a global planner with visual odometry.

Digital Twin Campus in Gazebo

Gazebo Digital Twin Validation

To enable safe, repeatable, and low-cost testing of the autonomous system, we developed a custom Gazebo simulation environment that precisely replicates our university campus.

This simulated world is heavily utilized to test and validate Automama’s full autonomous stack, including point-to-point navigation, control, and perception modules prior to real-world execution.

RL_Race: Custom PPO Training Architecture

PPO ROS 2 Ignition Gazebo End-to-End RL

Implemented a fully custom Proximal Policy Optimization (PPO) training pipeline for autonomous racing. The system uses end-to-end neural networks for actor/critic models without relying on pre-built libraries like Stable Baselines.

Continuous visual and LIDAR-based state representations allow the agent to learn optimal strategies, while integrated SLAM enables automatic track mapping.

>> PHASE 3: PERCEPTION & REAL-WORLD DEPLOYMENT

F

Perception Stack

YOLOv8n-Seg Stereo Depth IMX219 Occupancy Grid

The perception stack combines stereo vision and deep learning. A YOLOv8n-Seg model processes the left feed to identify road regions, while stereo depth estimation generates distance matrices for these segmented areas.

Data is projected into a 2D occupancy grid, enabling the vehicle to detect free space and plan paths by following the largest gap. This seamlessly integrates with visual odometry for global mapping.

🔗 View Perception Stack
G

Navigation Stack Real World Deployment

Zero-Copy Memory Potential Fields GPU Accel

Engineered for real-time decision-making with GPU-accelerated processing. Using DMA and zero-copy memory, sensor data streams efficiently to the GPU for parallel computation.

A potential fields method guides trajectory generation (attraction to goal, repulsion from obstacles). Hard vehicle kinematic and dynamic constraints are applied so the planned trajectory remains physically feasible.
