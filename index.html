Project Bee (2025 – Present)

Autonomous UAV for GPS-Denied exploration and surveillance

PPO Visual SLAM Edge AI Deployment HIL HRI

Developing a prototype framework for a generalized Robotics-as-a-Service (RaaS) platform. Project Bee is an autonomous UAV engineered for localized charging-station tracking, spatial exploration, and object-centric world-modeling. The system exposes the drone's real-time mental map and automatically extracted landmarks to the operator, enabling intuitive text or voice commands to execute dynamic, situational inspection tasks.

  • Autonomous Spatial Exploration: Complete spatial mapping of unknown indoor environments paired with native, localized return-to-charge mechanics.
  • Semantic World Modeling: Real-time translation of physical objects into localized landmarks, populating a 3D map with shared context both the machine and human understand.
  • Natural Language Control Stack: Intuitive human-in-the-loop interface translating text or voice prompts (e.g., area inspections, live feed routing) directly into autonomous flight execution via mapped keywords.

Project Automama (2023 – 2025)

Level-2 Autonomous Ground Vehicle (AGV)

ROS2 CUDA/TensorRT Jetson Orin Nano L2 Autonomy

An autonomous ground vehicle (AGV) built on a ROS2 architecture, executing perception and control pipelines on edge GPU hardware. The system operates at a sustained full operational cycle of 12.33 FPS in real-time, autonomously managing steering, throttle, and braking based on live environmental data.

  • Edge Perception: Executes zero-shot object detection and continuous road segmentation using CUDA, CuPy, VPI and TensorRT without off-board compute deployed on Jetson Orin Nano (8 GB).
  • Autonomous Navigation: Real-time path planning and dynamic obstacle avoidance integrated with ESP32-based hardware controllers and sensor integration for syncronized sensor snap cycles.
  • Hardware Architecture: Custom-fabricated mechanical chassis equipped with an isolated power management stack to support continuous GPU & Motor load.

Project ORCA (2023 – 2025)

Semi-Autonomous Tethered ROV for Marine Exploration

ROS2 Jetson Orin Tethered ROV SAUVC 2025

Served as Project Lead for an underwater platform that reached the semi-finals of the Singapore AUV Challenge (SAUVC) 2025. Transitioned the design from CAD to a fully functional tethered ROV featuring Ethernet-based control and telemetry feedback, capable of executing semi-autonomous sub-tasks.

  • Hardware Architecture: Designed a waterproof pressure hull with signal-isolated, multi-layered PCBs. Integrated a Jetson Orin edge computer with ESP microcontrollers for direct sensor and motor actuation.
  • Semi-Autonomous Control: Engineered altitude and position-holding sub-routines utilizing gimbal-axis sonar readings for localized object referencing against hard surfaces.
  • State Estimation: Implemented ROS2 nodes for photometric odometry and pose estimation in conditionally constrained underwater environments.
  • Research Funding: Secured project grants from the University Innovation Hub (World Bank funded) for marine surveillance R&D.
